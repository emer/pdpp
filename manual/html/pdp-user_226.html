15.6 Cs Defaults

The following default files (see section 9.5 Customization Through Defaults and Settings) are available for configuring different versions of the Cs objects:

`cs.def'
This is the standard defaults file, which uses a SigmoidUnitSpec, and the standard process heirarchy, assuming stochastic processing.
`cs_det.def'
This is for deterministic Cs networks, which means that only one sample is made of activation states for computing weight changes at the end of settling, and the CsMaxDa settling stat is created by default.
`cs_prob_env.def'
This configures the probabalistic output patterns version of Cs, meaning that the CsSettle process is created, and the environment will create probablistic events (see section 15.7 The Probability Environment and Cs).
