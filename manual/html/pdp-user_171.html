12.8 Different Types of Statistics

There are a number of built-in statistic types that come with the software. They are as follows:

  • 1 Summed-Error Statistics
  • 2 Monitoring Network State Variables
  • 3 Finding The Event Closest to the Current Output Pattern
  • 4 Comparing or Computing on Stat Values
  • 5 Activity-based Receptive Fields
  • 6 Reaction-time Based on Crossing an Activation Threshold
  • 7 Statistics that Provide Counters (Time, Epoch, etc)
  • 8 Miscellaneous Other Stat Types
