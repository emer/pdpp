14.2.8 RBp Defaults

The following default files (see section 9.5 Customization Through Defaults and Settings) are available for configuring different versions of the RBp objects:

`rbp.def'
This is the standard defaults file.
`rbp_ap.def'
This is for doing Almeida-Pineda (see section 14.2.7 The Almeida-Pineda Algorithm in RBp) in RBp.
