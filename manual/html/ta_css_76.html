8.3.3 Array Editing

Arrays are editing using an enhanced version of the standard Edit Dialog See section 7.6 The Edit Dialog. The Array Edit Dialog arranges all of the elements of the array horizontally and allows the user to scroll though the array elements using a horizontal scroll bar at the bottom of the dialog. In addition, number arrays (floats, ints, etc..) can be edited using a Color Array Edit Dialog which is similar in layout to the Array Edit Dialog, but adds a color palette for painting the values See section 7.18 Color Scale Specifications.
