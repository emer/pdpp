7 Guide to the Graphical User Interface (GUI)

This chapter provides a general guide and reference for using the graphical user interface to the PDP++ software. This covers all of the generic aspects of the interface--details about specific parts of the interface like the network viewer are found in the the section of the manual that covers the object in question (e.g., @xref{net-view}).
