7.16.2 Edit Dialog Menus

At the top of the edit dialog is a menubar with at least two menus. The first menu entitled Object is similar to the Object menu on a WinBase window. If the edited object is not a WinBase, the "Object" menu my only provide the "Load", "Save, "Save As", and "Close" menu choices. (see section 7.3 The "Object" Menu). The second menu called "Actions" may contain member functions particular to the edited object's type which will be called on the edited object.
